Windows Hotkeys

Ctrl + c

Copy selected content

Ctrl + v

Paste copied content

Ctrl + x

Cut selected content (Remove from current location with the possibility to place it elsewhere)

Ctrl + Shift + d

Duplicate the line where the cursor is placed or the selected part

Ctrl + z

Undo the previous action


Need for Use

Using key combinations allows you to simplify the execution of constantly repeated or simple actions that require a couple or more clicks (left and right mouse button presses), which significantly increases the number of actions performed with the keyboard.

Using Key Combinations

As an example, let's consider the key combination Ctrl+Shift+d.

You should not press the keys sequentially (like in calculators) or simultaneously (like playing a piano chord). First, you need to press and hold the Ctrl and Shift keys, and without releasing them, press the d key. The Ctrl and Shift keys held at the moment of pressing d modify the action of the d key. That is, when pressing the d key, the primary action (typing the d character) will not be executed, but one of the "secondary" actions (Duplication) will be performed.
